# 🎓 Student Management System

A simple Python-based Student Management System designed to manage student records efficiently.

## ✨ Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save Records in a Text File

## 🛠 Technologies Used

- Python
- File Handling
- Functions

## 📂 Project Structure

Student-Management-System/
│── main.py
│── students.txt
│── requirements.txt
│── README.md

## ▶️ How to Run

1. Install Python.
2. Download or clone this repository.
3. Run:

```bash
python main.py
```
